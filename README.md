# 🧗 BlocBoard

**BlocBoard** est une application mobile développée pour la salle **Boulder Line**.  
Elle permet de consulter, gérer et suivre les ouvertures d’escalade en bloc.

---

## 🚀 Stack technique
- **Frontend :** React Native (Expo)
- **Backend :** Supabase (PostgreSQL, Auth, Storage)
- **Langage :** TypeScript
- **Gestion de version :** Git + GitHub

---

## 📦 Installation

```bash
git clone https://github.com/ClementFourment/BlocBoard.git
cd BlocBoard
npm install
npm start
